# Splitwise Clone

This is a simple Splitwise clone project with frontend and backend.

## Features

- Add and split expenses
- Create and manage groups
- View user and group balances

## Tech Stack

- Frontend: ReactJS, Tailwind CSS
- Backend: FastAPI (Python)
- Database: SQLite
